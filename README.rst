Flake8-JSON
===========

This is a plugin for Flake8 that will emulate pylint json output.

Installation
------------

.. code-block:: bash

    pip install 


Usage
-----

.. code-block:: bash

    flake8 --format=json ...


Competitors
-----------

None that I could find on PyPI
